# 小炸世界 - 時代少年團粉絲社群平台

## 專案概述
歡迎蒞臨小炸世界(黃/黑調)
---- 時代少年團永遠在一起 ----

### 平台定位
- 專為時代少年團粉絲打造的綜合性社群平台
- 整合資訊分享、代購服務、社群互動、後台管理等功能
- 提供安全、便捷的粉絲交流與購物體驗

## 用戶系統

### 註冊功能
- **成為大米爆的一員**
  - 自定義ID(英文)、暱稱(中文)
  - 屬性選擇(團/團偏/唯)


### 登入功能
- **已經是大米爆**
  - 支援ID或暱稱登入
  - 記住登入狀態

### 訪客模式
- **先逛逛**
  - 可瀏覽公開內容
  - 無法參與互動功能
  - 引導註冊提示

## 代購平台

### 進入權限
- 進入代購區需輸入密碼
- 請先完成基本資料
- 標示匯率計價
- 付款資訊

### 商品分類
- **官製商品**
  - 訂單代碼:sdfj+西元2碼+月日+流水4碼
- **平台商品**
  - 訂單代碼:pt+西元2碼+月日+流水4碼
- **飯製商品**
  - 訂單代碼:fzh+西元2碼+月日+流水4碼

### 下單流程(許願)
- 商品瀏覽
- 加入購物車
- 下單，取得付款資料
- 訂單追蹤

## 資料庫關聯關係與索引設計

### 主要關聯關係
1. **Users** ↔ **UserLoginLogs**: 一對多 (UserCode)
2. **Users** ↔ **UserPermissions**: 一對多 (UserCode)

### 常用查詢範例

#### 用戶相關查詢
```sql
-- 查詢活躍用戶
SELECT UserCode, NickName, LastLoginDate, Status
FROM Users
WHERE Status = '啟用' AND LastLoginDate >= DATEADD(day, -30, GETDATE());

-- 查詢用戶登入記錄
SELECT u.NickName, l.LoginTime, l.IPAddress, l.LoginStatus
FROM Users u
JOIN UserLoginLogs l ON u.UserCode = l.UserCode
WHERE l.LoginTime >= DATEADD(day, -7, GETDATE())
ORDER BY l.LoginTime DESC;
```

#### 內容相關查詢
```sql
-- 查詢所有成員資料
SELECT MemberName, MemberNameEn, Birthday, FanName, SupportColor
FROM Members
WHERE IsActive = 1
ORDER BY SortOrder;

-- 查詢歌曲專區內容
SELECT SectionTitle, Content
FROM MusicSection
WHERE IsActive = 1
ORDER BY SortOrder;

-- 查詢演唱會專區內容
SELECT SectionTitle, Content
FROM ConcertSection
WHERE IsActive = 1
ORDER BY SortOrder;

-- 查詢綜藝節目內容
SELECT SectionTitle, Content
FROM VarietySection
WHERE IsActive = 1
ORDER BY SortOrder;
```

#### 統計查詢
```sql
-- 用戶註冊趨勢
SELECT
    CAST(JoinDate AS DATE) AS RegisterDate,
    COUNT(*) AS NewUsers
FROM Users
WHERE JoinDate >= DATEADD(month, -6, GETDATE())
GROUP BY CAST(JoinDate AS DATE)
ORDER BY RegisterDate;

-- 內容專區統計
SELECT
    'MusicSection' AS TableName,
    COUNT(*) AS RecordCount
FROM MusicSection
WHERE IsActive = 1
UNION ALL
SELECT
    'ConcertSection' AS TableName,
    COUNT(*) AS RecordCount
FROM ConcertSection
WHERE IsActive = 1
UNION ALL
SELECT
    'VarietySection' AS TableName,
    COUNT(*) AS RecordCount
FROM VarietySection
WHERE IsActive = 1;
```
